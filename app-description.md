## 📝 应用商店详细描述

### 应用简介

**分贝监测**是一款简洁、专业的环境噪音监测工具，专为需要长时间监测环境噪音的用户设计。无论是监测睡眠环境、办公室噪音，还是评估居住环境质量，分贝监测都能为您提供准确、详细的数据支持。

### ✨ 核心功能

**🎯 实时监测**
- 实时显示当前环境分贝值
- 直观的颜色编码：绿色（安静）→ 黄色 → 橙色 → 红色（嘈杂）
- 动态状态指示，一目了然

**🌙 后台持续监测**
- 支持整晚后台运行，即使屏幕关闭也能持续监测
- 通知栏实时显示当前分贝值
- 最长支持 8 小时连续监测

**📊 数据统计与分析**
- 实时显示最小值、平均值、最大值
- 按日期查看历史监测记录
- 近 7 天趋势图表，直观展示噪音变化
- 详细的会话报告，包含分贝分布和时间曲线

**📈 历史报告**
- 按日期浏览所有监测会话
- 每日趋势概览，快速了解噪音情况
- 点击查看详细的单日分析报告

**⏰ 定时停止**
- 设置定时停止时间，自动结束监测
- 适合睡前启动，早晨自动停止

**🎨 现代化设计**
- 基于 Material Design 3 设计规范
- 简洁直观的用户界面
- 流畅的动画效果
- 支持深色模式（跟随系统）

### 🔒 隐私与安全

**完全本地化**
- 所有数据存储在您的设备本地
- 不上传任何数据到服务器
- 不收集任何个人信息
- 不包含任何第三方分析或广告 SDK

**权限透明**
- 仅需要麦克风权限用于采集环境声音
- 通知权限用于后台监测时显示状态
- 不录制或存储原始音频，仅计算分贝数值

### 🎯 适用场景

✅ **睡眠环境监测**  
了解夜间噪音情况，确保卧室环境符合甲级标准（夜间≤30dB），改善睡眠质量

✅ **办公环境评估**  
监测工作环境噪音，理想办公环境应保持在40-50dB，提高工作效率和专注力

✅ **居住环境检测**  
评估住宅周边噪音水平，确保符合国家标准（1类区昼间≤55dB，夜间≤45dB）

✅ **学习环境测试**  
确保安静的学习氛围，理想学习环境应保持在30-40dB

✅ **听力健康保护**  
长期暴露在60dB以上环境需注意防护，超过85dB建议佩戴护耳设备

✅ **噪音投诉取证**  
记录噪音数据作为参考，为维权提供数据支持  

### 💡 使用建议

1. **首次使用**：授予麦克风权限，关闭电池优化以确保后台运行
2. **睡眠监测**：睡前启动监测，设置定时停止时间（如早晨 7:00）
3. **充电使用**：长时间监测建议连接充电器
